# トランスクリプトから議事録作成API 実装計画書

## 1. プロジェクト概要

### 1.1 目的
音声トランスクリプトから議事録を自動生成するAPIシステムの構築

### 1.2 主要機能
- トランスクリプトから議事録への変換（OpenAI API使用）
- JWT認証によるユーザー認証システム
- ユーザー管理（DB保存）
- ログ機能
- 環境変数による設定管理

## 2. システム構成

### 2.1 ディレクトリ構造
```
project/
├── routers/           # APIルーティング
│   ├── __init__.py
│   ├── auth.py        # 認証関連エンドポイント
│   ├── minutes.py     # 議事録生成エンドポイント
│   └── users.py       # ユーザー管理エンドポイント
├── modules/           # ビジネスロジック
│   ├── __init__.py
│   ├── auth.py        # JWT認証処理
│   ├── database.py    # データベース接続・操作
│   ├── openai_client.py # OpenAI API連携
│   └── logger.py      # ログ機能
├── schemas/           # データモデル定義
│   ├── __init__.py
│   ├── auth.py        # 認証関連スキーマ
│   ├── minutes.py     # 議事録関連スキーマ
│   └── users.py       # ユーザー関連スキーマ
├── main.py           # FastAPIアプリケーション
├── requirements.txt  # 依存関係
├── .env.example     # 環境変数テンプレート
└── README.md        # プロジェクト説明
```

### 2.2 技術スタック
- **フレームワーク**: FastAPI
- **認証**: JWT (JSON Web Token)
- **データベース**: SQLite（開発用）/ PostgreSQL（本番用）
- **AI API**: OpenAI GPT API
- **ログ**: Python標準ログライブラリ
- **環境管理**: python-dotenv

### 3.3 APIエンドポイント設計

#### 3.3.1 認証エンドポイント
```
POST /auth/register    # ユーザー登録
POST /auth/login       # ログイン
POST /auth/refresh     # トークンリフレッシュ
GET  /auth/me          # ユーザー情報取得
```

#### 3.3.2 議事録エンドポイント
```
POST /minutes/generate    # 議事録生成
GET  /minutes/           # 議事録一覧取得
GET  /minutes/{id}       # 特定議事録取得
DELETE /minutes/{id}     # 議事録削除
```

#### 3.3.3 ユーザー管理エンドポイント
```
GET  /users/profile      # プロフィール取得
PUT  /users/profile      # プロフィール更新
```

### 3.4 セキュリティ設計

#### 3.4.1 パスワード管理
- bcryptによるハッシュ化
- ソルト付きハッシュ

#### 3.4.2 JWT設定
- アクセストークン有効期限: 15分
- リフレッシュトークン有効期限: 7日
- HS256アルゴリズム使用

#### 3.4.3 API保護
- 全ての議事録関連エンドポイントでJWT認証必須
- レート制限の実装

### 3.5 ログ設計

#### 3.5.1 ログレベル
- DEBUG: 開発時のデバッグ情報
- INFO: 一般的な処理情報
- WARNING: 警告レベルの問題
- ERROR: エラー情報
- CRITICAL: 致命的なエラー

#### 3.5.2 ログ出力項目
- タイムスタンプ
- ログレベル
- ユーザーID（認証済みの場合）
- エンドポイント
- 処理時間
- エラー詳細（エラー時）

### 3.6 環境変数設計

```bash
# .env
# データベース設定
DATABASE_URL=sqlite:///./app.db

# JWT設定
JWT_SECRET_KEY=your-secret-key-here
JWT_ALGORITHM=HS256
JWT_ACCESS_TOKEN_EXPIRE_MINUTES=15
JWT_REFRESH_TOKEN_EXPIRE_DAYS=7

# OpenAI設定
OPENAI_API_KEY=your-openai-api-key-here
OPENAI_MODEL=gpt-3.5-turbo

# ログ設定
LOG_LEVEL=INFO
LOG_FILE=app.log

# アプリケーション設定
APP_NAME=Minutes Generator API
APP_VERSION=1.0.0
DEBUG=False
```
